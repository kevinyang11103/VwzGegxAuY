# 学生考勤管理系统 python1180

## 项目概述

学生考勤管理系统是一款基于 Python 和 Django 框架开发的 Web 应用程序，旨在帮助学校管理学生的考勤情况。系统集成了首页、班级管理、教师管理、学生管理等功能模块，实现了考勤信息的高效管理。

## 技术栈

### 后端

- Python 3.8.5
- Django
- MySQL 5.7 / MySQL 8

### 前端

- Vue
- HTML
- JavaScript

## 功能模块

1. 首页
2. 班级管理
3. 教师管理
4. 学生管理
5. 科目信息管理
6. 课程信息管理
7. 签到记录
8. 请假记录
9. 考勤管理
10. 考勤统计
11. 系统管理
12. 用户信息管理

## 系统角色

- 管理员：负责系统管理和用户管理
- 教师：负责班级、学生考勤信息管理
- 学生：查看个人考勤信息
